1. Introduction
----------------

Le mod PNJ Tiax pour Baldur's Gate II (Tiax pour faire court) est un module pour le jeu Baldur's Gate II, avec ou sans l'extension Throne of Bhaal, pour lequel du contenu est présent.

Tiax, un personnage non-joueur initialement apparu dans le premier jeu de la série des Baldur's Gate, est un gnome voleur/clerc, un prêtre de Cyric qui n'a qu'une seule idée en tête : dominer le monde. Citation la plus célèbre : "Tiax règne sur tout !"

----------------
2. Installation
----------------

Assurez-vous d'avoir téléchargé et installé le patch Bioware le plus récent.
Installez le patch officiel AVANT toute modification non officielle.

Assurez-vous que Baldur's Gate II et que tout programme utilitaire apparenté ne soit pas ouvert avant d'installer Tiax. Ceci inclut les programmes tels que Shadowkeeper et Infinity Explorer.

Pour installer le mod, lancez Tiax_v4.exe. 

Une fois que le Readme est affiché, l'installation est terminée.

Pour réinstaller ou désinstaller le mod, lancez Setup-Tiax.exe dans le fichier d'installation du jeu.

-------------------------
3. Informations générales
-------------------------

Tiax peut se joindre au groupe dans le Chapitre 4. Il n'aura aucun équipement, il est donc conseillé d'en prévoir pour lui à l'avance.

Tiax a une série de dialogues "d'amitié" avec le PJ, des dialogues avec tous les PNJ de Bioware, un certain nombre d'interactions avec les personnages rencontrés, des réactions aux évènements et d'autres dialogues en fonction de l'environnement.

selon le niveau du PJ dans le Chapitre 4, l'expérience de Tiax peut être de 400000, 800000, 1200000, 1600000 ou 2000000 points d'expérience.

Dans ToB, Tiax démarre avec 2500000 points d'expérience. Lorsqu'il atteindra le 25ème niveau dans la classe de clerc, il obtiendra le Symbole Sacré de Cyric.

Si vous avez des problèmes avec les sons de sélection/confirmation de Tiax, sélectionnez-le dans le jeu et appuyez sur la touche K. Cela devrait régler le problème.

-------------------------------
4. Questions Fréquemment Posées
-------------------------------

Q: Où est Tiax?
A: La réponse à cette question dévoilera des éléments de l'histoire du jeu.

Vous rencontrerez Tiax dans le Chapitre 4, dans Spellhold, après le départ d'Irenicus, lorsque les détenus tombent raides morts. Vous pouvez aussi le recruter à partir de l'Esprit du Destin dans Throne of Bhaal.

Q: Mais il n'y est pas !
A: Assurez-vous d'avoir bien installé le mod. Ouvrez le fichier d'installation du jeu et lancez Setup-Tiax.exe. Si le fichier n'y est pas, reprenez le processus d'installation comme décrit ci-dessus.

Q: Les sons de sélection/confirmation de Tiax sont complètement mélangés !
A: Sélectionnez Tiax et appuyer sur la touche K.

Q: Tous les personnages se trompent de dialogue !
A: Essayez de démarrer une nouvelle partie.
Q: Cela n'a pas résolu le problème !
A: Refaites une installation propre, réinstallez vos mods et débutez une nouvelle partie.

Q: Tiax parle dans SoA, mais pas dans ToB !
A: Ouvrez le fichier d'installation du jeu, et lancez Setup-Tiax.exe. Vous serez obligé de charger à nouveau votre partie SoA, par contre. 
Q: Cela n'a pas résolu le problème !
A: Refaites une installation propre, réinstallez vos mods et débutez une nouvelle partie.

Q: Est-ce que Tiax est compatible avec le mod "XYZ" ?
A: Tiax devrait fonctionner avec les autres mods. Référez-vous au Readme du mod afin d'obtenir des informations supplémentaires ainsi que des conseils sur l'ordre d'installation des mods.
